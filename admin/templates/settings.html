Authentication

—

Configured via ./install.sh --enable-auth. The panel logs in via PAM (your Pi user). Streams are gated by a shared stream:<password> credential — rotate it here without touching the installer.

WebRTC public access

—

mediamtx advertises ICE host candidates so external WebRTC viewers know how to reach this Pi. Without a routable public IP, only LAN clients connect. The panel auto-detects via curl ifconfig.me on boot and refreshes on a timer; set a DDNS hostname or a static IP below to override.
